Schlüsselwörter:
	- break (Vorzeitiger Abbruch einer Schleife; beendet die Ausführung eines case-falls)
	- bool (Datentype für die möglichen Werte true und false)
	- case (Zu behandelnder Fall im Zusammspiel mit switch)
	- char (Datentyp für ASCII-Zeichen; für ASCII-Zeichen -->unsigend<-- char)
	- const (macht eine Variable unveränderbar bzw. macht eine Variable zu einer Konstanten)
	- continue (Erzwingt den nächsten Schleifendurchgang bzw. Schleifeniteration)
	- default (Alternativer Fall, falls kein "case-fall" zutrifft)
	- do (wird von while benötigt; fußgesteuerte do ... while () - Schleife)
	- double (Datentyp)
	- else (leitet den alternativ auszuführenden Codeblock einer if-Anweisung ein)
	- float (Datentyp)
	- for (leitet die Formulierung einer for-Schleife ein)
	- if (Leitet die Abfrage einer Bedingung ein; falls erfüllt, wird die Konsequente
		ausgeführt; falls nicht, die Alternative)
	- int (Datentyp)
	- long (Datentyp)
	- return (Kehre aus einer Funktion an die aufrufende Codestelle zurück; bei dieser Gelegeheit kann/muss auch ein Wert zurückgegeben werden)
	- signed (im Zusammenspiel mit Variablen/Datentypen -> KEIN Verzicht auf negative Werte)
	- struct (Anlegen eines selbst definierten Datentyps)
	- switch (Leitet eine Fallunterscheidungs-Kaskade ein)
	- unsigned (im Zusammenspiel mit Variablen/Datentypen -> Verzicht auf negative Werte)
	- void (Gibt im Zusammenhang mit Funktionen den Rückgabetyp "nichts" an)
	- while (leitet die Formulierung einer while-Schleife ein)
